HOW TO PUSH AND PULL
-add files to commit log
	git add/rm <file> [-a for all]
-commit changes
	git commit [-a for all] [-m "memo"]
-sync to GitHub server
	git push
	requires PAT or other 2FA
-sync from GitHub server
	git pull
